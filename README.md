# FlowThrough_CoastalAg

This repository contains open-source R code, data, and supporting files used to process Membrane Inlet Mass Spectrometry (MIMS) data collected during continuous flow soil incubation experiments. 

## Overview 

**Aim**: Develop a reproducible workflow for processing raw MIMS output into dissolved gas concentrations and gas fluxes for continuous flow soil incubation experiments. 

 

## Repository Structure 

**MIMS_calculations**: Imports raw MIMS output, applies gas solubility equations then calculates dissolved gas concentrations and gas fluxes. 

**analyses**: Contains R scripts that calculate average flow rates, analyzes Teros 12 soil sensor data, and statistically analyze core flux data. 

**csv_out**: Contains output files generated from R script. 

**data**: Contains Hyde County (MRW) and Pitt County (YHR) supporting data files.

**figures**: Contains figures generated from processed datasets.
